# Infra BWX

Android app for reporting public infrastructure issues in Banyuwangi Regency. Built with Kotlin and Jetpack Compose.

## Features

- In-app camera to capture road damage, uneven patches, and street light outages
- Auto GPS location and sub-district detection
- Top 5 ranking per category from submitted reports
- Submit reports directly to a Google Apps Script backend (images stored in Drive, metadata in Sheets)

## Tech Stack

- **Android:** Kotlin, Jetpack Compose, CameraX, MVVM, Retrofit, DataStore
- **Backend:** Google Apps Script (free, no cloud costs)
- **Min SDK:** 31 (Android 12) | **Target:** 34
